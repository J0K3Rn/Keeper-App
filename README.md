# Keeper-App
 
React Styled Google Keep App

Features:
- MUI for extra design functionality
- React states to track notes

Run the app:
- Download repository
- Open downloaded repository with a command line interface
- cd into Keeper-App/keeper-app
- run `npm install`
- run `npm start`
